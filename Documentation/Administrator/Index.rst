Administrator Manual
====================

Target group: **Administrators**

.. _admin-installation:

Installation
------------

This extension does not have any major requirements

- PHP 5.3-5.6

- TYPO3 CMS 6.2 (It is known to run with version 7.5, but this is unsupported)

- TYPO3 should obviously be able to access the internet

Just install it from the TER and continue on to the section "Configuration"


.. _admin-configuration:

Configuration
-------------

Some basic configuration needs to be done:

- You need an AppID / API Key from `openweathermap.org`_. Enter this AppID in the extension configuration inside the extension manager.

- Optionally change the cache lifetime in the extension configuration.

.. _openweathermap.org: http://openweathermap.org/
